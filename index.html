Accessibility Issues in Ad Driven Web Applications

Artifact for ICSE 2025

Project Overview

This artifact supports our paper exploring the impact of third-party advertisements on web accessibility. It enables you to replicate our experiments by using a pre-configured Virtual Machine that includes:

Release (v2.0.1): Accessibility-Issues-in-Ad-Driven-Web-Applications(v2.0.1)

  • A complete source code and dataset setup
  • Automated scripts to crawl websites and evaluate the accessibility of ads
  • Tools for reproducing experimental results and generating figures

By comparing websites with and without ads, the artifact helps identify common accessibility (WCAG) violations—such as issues with focus management and non-descriptive elements—introduced by ad-driven content.

Artifact Details

Installation & Usage

Detailed installation instructions—including steps to import the VM into VMware Fusion, activate the Conda environment, and run the necessary scripts—are available in our README file.

In summary, the artifact allows you to:

  • Crawl websites with and without ads to identify accessibility violations
  • Reproduce the experimental results and figures presented in our paper
  • Explore and analyze the impact of ad-driven web applications on accessibility
